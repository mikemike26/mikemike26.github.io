Michigan Master Anglers

This is a for fun app I did to get my head wrapped around leafletJS and to familiarize myself with geospatial development. I hit the open data api from data.MICHIGAN.gov and pulled down the list of master angler submissions in michigan and then hit the google maps api to get the coordinate data for each city listed. I then mapped the locations with leaflet and then generated different sized SVG circles depending on how many submissions were made in each city. You can filter the map by fish species by clicking the checkboxes and then if you click on a city, the list of anger info will be shown on the right.

Beards Brewery

The folks at Beards Brewery in Petoskey, Michigan didn’t really have much to go on branding wise besides an awesome logo. We created a user experience that showcases their fun loving personalities and passion for making great beer. Oh... and we included a lot of facial hair as well; always a good detail.


  • breards-brewery.com

Their site is built with Ruby on Rails using RefineryCMS which allows them to change and add content easily from a back-end interface. This site includes a fully integrated blog to add to the SEO goodness and it’s set up to accept an ecommerce integration with Spree in the future. To top it off, the site has a responsive layout which allows it to look great and work well on all sizes of screens, including tablets and smartphones.

Intentional Health Products

Intentional Health Products is a small business specializing in natural medicines and remedies. This site is receiving a face lift and a new back end built with Ruby on Rails using RefineryCMS. The gradients and text shadows were removed and a new layout was created to look more like a small business rather than a blog (this was an old self hosted Wordpress site). The site includes a blog and a responsive layout.

Epicurean Essentials

Epicurean Essentials is an ecommerce site built with Ruby on Rails using Spree. This features a full cms back-end that allows them to manage all of their payment information, inventory, and web content.

Savory Celebration
